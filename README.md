## Progressive Web App (PWA)

This application is a fully functional PWA with the following capabilities:

- **Installable**: Install on desktop, mobile, and tablet devices
- **Offline Support**: Full functionality works offline after initial load
- **Auto Updates**: Automatically updates in the background
- **Native Feel**: Launches from home screen like a native app
- **Fast Loading**: Cached resources for instant loading

### Installing as PWA

1. Visit the website in a supported browser (Chrome, Edge, Safari, Firefox)
2. Look for the install prompt or use the browser's install option
3. On mobile: Tap the share button and select "Add to Home Screen"
4. On desktop: Click the install icon in the address bar

## Run Locally

**Prerequisites:** Node.js and Yarn (or npm)


1. Install dependencies:
   ```bash
   yarn install
   ```

2. Run the development server:
   ```bash
   yarn dev
   ```

3. Build for production:
   ```bash
   yarn build
   ```

4. Preview production build:
   ```bash
   yarn preview
   ```
